== Tag images

A small application that helps tagging images.

Tags can then be exported through various rake tasks.

=== Tagging images

Start tagging by visiting /images.

=== View tagged images

Go to /images/overview?show_tagged=true to see tagged images. By default images with the tag 'OK' are shown, but you can specify your flag of interest using the url /images/overview?show_tagged=true&tag=your_tag.
